💻 Tutorial de Java

Proyecto básico de programación

Ciclos

Sirven para repetir instrucciones varias veces.

for(int i=1;i<=5;i++){ System.out.println(i); }

Métodos

Bloques de código que realizan una tarea.

public static void hola(){ System.out.println("Hola"); }

Vectores

Guardan varios datos en una sola variable.

int[] v = {1,2,3};

Listas

Permiten guardar datos dinámicos.

ArrayList lista = new ArrayList<>(); lista.add("Hola");

Archivos

Sirve para guardar información.

File f = new File("datos.txt");

# 📚 Conceptos Fundamentales
🔄 Ciclos (Bucles) ## ¿Qué es un ciclo? Un ciclo permite repetir instrucciones varias veces sin escribir el mismo código repetidamente. ## ¿Para qué sirve? - Automatizar tareas. - Reducir código. - Recorrer datos. ## Ejemplo ```java for(int i=1; i<=5; i++){ System.out.println(i); } ``` ## Explicación - `i=1` inicia el contador. - `i<=5` es la condición. - `i++` incrementa el valor. ## Ejercicio Mostrar los números del 1 al 10. ## Solución ```java for(int i=1; i<=10; i++){ System.out.println(i); } ```
---
⚙️ Métodos (Funciones) ## ¿Qué es un método? Un método es un bloque de código reutilizable. ## ¿Para qué sirve? - Organizar programas. - Evitar repetir código. ## Ejemplo ```java public static int suma(int a, int b){ return a+b; } ``` ## Ejercicio Crear un método que multiplique dos números. ## Solución ```java public static int multiplicar(int a, int b){ return a*b; } ```
---
📊 Vectores y Matrices ## ¿Qué es un vector? Es una estructura que almacena varios datos del mismo tipo. ```java int[] numeros = {1,2,3,4,5}; ``` ## ¿Qué es una matriz? Organiza datos en filas y columnas. ```java int[][] matriz = { {1,2}, {3,4} }; ``` ## Ejercicio Crear un vector con cinco nombres. ## Solución ```java String[] nombres = { "Ana", "Luis", "Carlos", "Pedro", "Maria" }; ```
---
📋 Listas ## ¿Qué es una lista? Una colección dinámica de datos. ## Ejemplo ```java ArrayList nombres = new ArrayList<>(); nombres.add("Ana"); nombres.add("Luis"); ``` ## Ventajas - Tamaño dinámico. - Fácil manipulación. ## Ejercicio Crear una lista con cinco ciudades.
---
📁 Manejo de Archivos ## ¿Qué es? Permite guardar y leer información desde archivos. ## Ejemplo ```java FileWriter archivo = new FileWriter("datos.txt"); archivo.write("Hola Mundo"); archivo.close(); ``` ## Ejercicio Crear un archivo llamado estudiantes.txt. ## Solución ```java archivo.write("Juan\n"); archivo.write("Maria\n"); archivo.write("Pedro\n"); ```

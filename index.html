💻 Tutorial de Programación en Java

Conceptos básicos explicados de forma sencilla

🔄 Ciclos (Bucles)

Los ciclos permiten repetir un bloque de código varias veces sin tener que escribirlo muchas veces. Son muy útiles cuando quieres automatizar tareas repetitivas.

¿Para qué sirven?

  • Repetir instrucciones
  • Recorrer listas o arreglos
  • Ahorrar código
for(int i=1;i<=5;i++){ System.out.println("Número: " + i); }

⚙️ Métodos

Un método es un bloque de código que realiza una tarea específica. Se usa para organizar mejor el programa y evitar repetir código.

¿Para qué sirven?

  • Reutilizar código
  • Organizar el programa
  • Hacer el código más limpio
public static void saludar(){ System.out.println("Hola, bienvenido"); }

📊 Vectores (Arreglos)

Un vector es una estructura que almacena varios datos del mismo tipo en una sola variable.

¿Para qué sirven?

  • Guardar múltiples valores
  • Acceder rápido a datos
  • Organizar información
int[] numeros = {10, 20, 30, 40}; System.out.println(numeros[0]);

📋 Listas

Las listas son estructuras dinámicas que pueden crecer o reducirse según lo que necesites.

¿Para qué sirven?

  • Guardar datos sin límite fijo
  • Agregar y eliminar elementos
  • Manejo de información flexible
ArrayList lista = new ArrayList<>(); lista.add("Juan"); lista.add("Maria");

📁 Archivos

El manejo de archivos permite guardar información en el computador y leerla después.

¿Para qué sirve?

  • Guardar datos permanentemente
  • Leer información guardada
  • Crear programas más avanzados
File archivo = new File("datos.txt");

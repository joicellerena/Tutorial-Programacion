🔄 Ciclos (Bucles)

Los ciclos permiten repetir instrucciones automáticamente varias veces sin escribir el mismo código repetidamente. 


for(int i = 1; i <= 5; i++){
    System.out.println(i);
}

⚙️ Métodos (Funciones)

Los métodos permiten reutilizar bloques de código para realizar tareas específicas. 


public static int suma(int a, int b){
    return a + b;
}

📊 Vectores y Matrices

Los vectores almacenan varios datos del mismo tipo y las matrices organizan datos en filas y columnas.

📋 Listas

Las listas permiten almacenar elementos dinámicamente.

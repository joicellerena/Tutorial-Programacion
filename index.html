💻 Tutorial de Programación en Java

Guía básica para aprender programación

🔄 Ciclos

Los ciclos permiten repetir acciones varias veces.

for(int i=1; i<=5; i++){ System.out.println(i); }

⚙️ Métodos

Un método es un bloque de código reutilizable.

public static void saludar(){ System.out.println("Hola mundo"); }

📊 Vectores

Los vectores almacenan varios datos en una sola variable.

int[] numeros = {1,2,3,4,5};

📋 Listas

Las listas permiten almacenar datos dinámicos.

ArrayList lista = new ArrayList<>(); lista.add("Hola");

📁 Archivos

Sirve para leer y escribir archivos.

File archivo = new File("datos.txt");

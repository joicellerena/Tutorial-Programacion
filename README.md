<h2 style="color:#5BC0BE;">🌟 ¿Por qué aprender Java?</h2>

<p style="font-size:20px;max-width:900px;margin:auto;text-align:justify;">
Java es uno de los lenguajes de programación más utilizados en el mundo. Se emplea en el desarrollo de aplicaciones empresariales, sistemas web, aplicaciones móviles Android, videojuegos y software de escritorio. Su facilidad de aprendizaje y gran comunidad lo convierten en una excelente opción para comenzar en el mundo de la programación.
</p>
